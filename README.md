# sodoff-serverside-mods
Server Side Mods created for the Sodoff Server Emulator

Includes fixes, price changes, and balancing. It also adds one new unique item that will give players a random dragon egg for those who are indecisive.

NOTICE: These mods will not work on the client, but will work on local hosted games (not connected to external servers like Sodoff/Project Edge) and other self-hosted servers.

I created these for both myself and my family to make the game more enjoyable with added challenge instead of getting infinite gems for free, which later led to finding all unadded items and creating obtainable versions that could be acquired via the shop.
